# 群聊创建向导第一步"下一步"无响应
> 创建时间: 2026-05-28 | 状态: 🟡排查中

## 问题描述
在聊天页面点击"创建群聊"按钮，弹出创建向导。填写任务描述后点击"下一步"，对话框关闭但没有创建会话。需要通过 @提及 选择 Agent 后才能创建群聊。

## 出现原因
待排查。可能的推测：
1. "下一步"按钮的 onClick 事件未正确触发创建流程
2. 创建流程需要 Agent 选择作为前置条件，但 UI 没有提示用户
3. 向导第一步和第二步（Agent 选择）的衔接逻辑有问题

## 解决方案
待排查。当前绕过方式：在聊天框输入 @Agent名 触发 Agent 选择面板，选择后点"创建群聊 (N 人)"。
