## 发现的Bug

### 🔴 Bug #1: API密钥未正确掩码
- **严重性**: 高（安全问题）
- **位置**: `GET /api/providers`
- **现象**: 5个Provider的API密钥以明文返回，未使用`maskApiKey()`处理
- **受影响Provider**: DupTest, DeepSeek, DouBaoSeed, MiniMax, Zhipu GLM
- **修复建议**: 确保所有Provider GET端点调用`maskApiKey()`函数

### 🟡 Bug #2: GBK编码乱码
- **严重性**: 中
- **位置**: Agent名称和描述的中文显示
- **现象**: "API测试Agent"显示为"API����Agent"，"前端工程师"描述显示乱码
- **原因**: Windows GBK编码与UTF-8不兼容
- **修复建议**: 确保所有API响应使用UTF-8编码，Content-Type包含`charset=utf-8`

### 🟡 Bug #3: Message API字段名不一致
- **严重性**: 低
- **位置**: `POST /api/sessions/{id}/messages`
- **现象**: 需要`rawContent`字段而非`content`，前端可能使用不同字段名
- **修复建议**: 统一字段命名或添加别名支持
